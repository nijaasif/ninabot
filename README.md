# NINA Bot – AI Hospital Assistant

NINA Bot is an AI-powered hospital assistant designed to automate patient interactions such as answering hospital-related queries and assisting with appointment workflows.

## Features
- AI chatbot for hospital information
- Appointment workflow support
- Frontend chatbot widget
- Knowledge-based response system

## Tech Stack
- Python
- HTML
- CSS
- JavaScript
- LLM / AI APIs

## Project Structure

frontend/ → Chatbot UI  
main.py → Backend logic  
hospital_knowledge.txt → Knowledge base  
appointments.csv → Appointment data  
requirements.txt → Dependencies
